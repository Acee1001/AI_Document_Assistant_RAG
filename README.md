# RAG Frontend

Next.js 14 + Tailwind chat UI for the FastAPI RAG pipeline backend.

## Setup

1. `cd rag-frontend`
2. `npm install`
3. `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)
5. Make sure the FastAPI backend is running at [http://127.0.0.1:8000](http://127.0.0.1:8000)

## Features

- Upload `.pdf` and `.txt` documents to the backend
- Ask questions and view AI responses with source chunks
- Dark-mode interface with responsive layout
- Graceful API/network error handling
